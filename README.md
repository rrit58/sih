#  DBT Connect – Unified Scholarship Awareness & Seeding Verification Platform

DBT Connect ek multilingual, AI-enabled awareness & verification platform hai jiska goal hai  
**scholarship access ko easy, transparent aur accessible banana.**

Is platform par students, citizens, schools, gram panchayats, NGOs, aur aanganwadi centers  
DBT seeding, scholarship information, awareness resources, aur campaign data manage kar sakte hain.

Platform pe **3 developers** ne saath milkar work kiya hai.

---

## Key Features Summary

- **Fully Multilingual Website (UI + Chatbot both)**
- **AI-based Multilingual Chatbot for help & guidance**
- **Analytics-based Home Dashboard (Live Stats)**
- **DBT Seeding Checker (Aadhar + Bank + Mobile verification)**
- **Nearest Centers Finder (Map + Distance Based Search)**
- **Google Maps Redirect for Navigation**
- **Scholarship Explorer (Category + Central/State Filters)**
- **Awareness Resources (Animated tutorials + Video lectures)**
- **Portal for NGOs / Panchayats / Schools / Aanganwadi**
- **Campaign Data Upload System (Photos, Videos, Awareness Reports)**
- **Login + Registration System for Centers**
- **Centralized backend API + secure database**

---

#  1. Home Dashboard – Real-Time Stats

Home page me live statistics show hota hai:

- Kitne students ne scholarships avail kiya  
- Category-wise data (SC/ST/OBC/General/etc.)
- Awareness percentage  
- Seeding gap analysis  
- Multilingual UI support  

Ye section visitors ko India me actual DBT awareness aur access ka quick snapshot deta hai.

---

# 2. Multilingual AI Chatbot

- Chatbot **multiple Indian languages** me available hai.  
- User koi bhi query pooch sakta hai:  
  - scholarship eligibility  
  - seeding status  
  - nearest centers  
  - portal ka use kaise kare  
- Natural language me guidance deta hai.

---

# 3. DBT Seeding Checker Dashboard

Is dashboard me user:

- Aadhaar number  
- Bank account number  
- Mobile number  

enter karke **seeded / not seeded** status check kar sakta hai.

### If NOT Seeded:
User ko automatically **Nearest Centers Dashboard** me redirect kiya jata hai taaki wo offline center me jaakar apni seeding complete kara sake.

---

# 4. Nearest Centers Finder + Map Integration

User yeh inputs dal sakta hai:

- Pin code  
- Category  
- District  
- Area  
- Current location (GPS)

System nearest centers show karta hai:

- Center name  
- Distance  
- Contact details  
- Opening hours  

### Navigation Support:
“**Open in Google Maps**” button se direct Google Maps pe redirect ho jaata hai navigation ke liye.

---

# 5. Scholarship Explorer Dashboard

Students apne:

- Category  
- State  
- Central scholarship  
- Eligibility criteria  

ke hisaab se scholarships explore kar sakte hain.

Har scholarship card me:

- Eligibility  
- Benefits  
- Required documents  
- Apply link  

clearly show hota hai.

---

# 6. Awareness Resources Dashboard

Yahaan user:

- Animated tutorials  
- Video lectures  
- Awareness sessions  

dekh sakta hai. Ye feature rural + first-time users ke liye helpful hai.

---

# 7. Our Portal – NGO, Panchayat, School & Aanganwadi Registration

Portal me 4 types of entities register kar sakti hain:

- NGO  
- School  
- Gram Panchayat  
- Aanganwadi  

### Registration Steps:

- Name + Category  
- Location  
- Official mobile number  
- Password creation  
- Login system (next time same password)

### Portal Features after Login:

- Conducted **awareness campaigns** upload  
- PICTURES upload  
- VIDEOS upload  
- DATA collected upload  
- Attendance / beneficiary count  
- Session details  

Ye data **government ko ground-level impact track** karne me help karta hai.
